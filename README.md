# Simulatore di Gara tra Cavalli

Un'applicazione Java che simula una gara tra cavalli, con eventi casuali di azzoppamento. I risultati vengono salvati in un file di testo.

## Descrizione

Il programma consente di simulare una gara tra 4 cavalli. Ogni cavallo ha un nome e una lentezza (in millisecondi tra i passi). Durante la gara, un thread separato pu√≤ azzoppare casualmente un cavallo, interrompendone la corsa. I progressi e l'esito della gara vengono registrati nel file `risultati_gara.txt`.
